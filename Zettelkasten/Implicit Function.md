### Definition:
The relation $f(x,y)=0$ defines $y$ as an implicit function of $x$ on the interval $I: a<x<b$, if there exists a function $g(x)$ defined on $I$ such that:

$$ f[x, g(x)] = 0 $$
for every $x$ in $I$.
