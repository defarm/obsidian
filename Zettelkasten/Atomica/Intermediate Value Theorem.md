### Statement:
Let $f:[a,b] \mapsto \mathbb{R}$ be [[Continuous on a Set]] and suppose that $f(a) < 0 < f(b)$ then there is a point $c$ such that $f(c) \in (a,b) = 0$. The [[proof of the intermediate value theorem]] is found here.
